## エグゼクティブサマリー

### プロダクト概要

**SmartOffice AI** は、AI技術を活用して中小企業のバックオフィス業務を統合的に効率化するSaaSプラットフォームです。請求書処理・契約書管理・議事録自動生成の3つのコア機能を単一プラットフォームで提供し、従来バラバラのツールで行っていた業務を一気通貫で自動化します。

### なぜ今か

1. **法制度の追い風**: 電子帳簿保存法の義務化・インボイス制度により、中小企業のデジタル対応が不可避
2. **AI技術の成熟**: Claude等の高性能LLMにより、日本語業務文書の高精度処理が実用レベルに到達
3. **深刻な人手不足**: 中小企業の68%が人手不足を実感、管理部門の自動化ニーズが急増

### 市場機会

- **TAM**: 2,700億円（AI×業務効率化SaaS市場全体）
- **SAM**: 614億円（中小企業×AI導入意向あり）
- **SOM**: 12.3億円（3年目ARR目標）
- 市場成長率: 年20〜25%（AI搭載SaaS領域）

### 差別化ポイント

- 請求書・契約書・議事録を **単一プラットフォーム** で提供（競合は単機能特化が主流）
- 各機能間のデータ連携によるシナジー（会議の決定事項→契約書作成→請求書発行の一気通貫フロー）
- 中小企業の価格感に合わせた段階的プライシング（無料〜月額98,000円）

- スタートアップ企業（従業員10名未満だが成長フェーズにある企業）

### 2.3 主要機能の技術的実現

#### 請求書処理フロー
```
アップロード → S3保存 → Textract(OCR) → Claude API(構造化抽出) → 検証・補正 → DB保存 → 通知
```
- 信頼度スコアで人間レビュー要否を判定
- バッチアップロード対応

#### 契約書管理フロー
```
PDFアップロード → テキスト抽出 → Claude API(要約・リスク分析・日付抽出) → OpenSearch登録 → DB保存 → 期限アラート設定
```
- テナント間の厳格なデータ分離
- 個人情報マスキング処理

#### 議事録自動生成フロー
```
音声アップロード → Transcribe(文字起こし+話者分離) → Claude API(整形・要約・アクションアイテム抽出) → テンプレート整形 → DB保存 → 共有通知
```
- 長時間会議は分割並列処理
- リアルタイム文字起こしはPhase 2で対応
